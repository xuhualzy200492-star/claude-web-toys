# claude-web-toys

两个纯前端、单文件、双击即用的网页小项目（与 Claude 一起 vibe coding 而成）。

## 点歌电台.html
交互式点歌电台：内置公开音乐电台直播频道，可调台收听；点歌模式支持拖入本地音乐 / 粘贴音频直链，自动连播。
带黑胶旋转、队列管理、搜索、音量进度控制。还集成了摄像头**手势控制**（握拳暂停 / 张掌播放 / 1指下一首 / 2指上一首 / 3指静音）。

## 手势粒子花.html
摄像头手势控制 3D 粒子花：MediaPipe 手部追踪 + Three.js 粒子引擎。
- 🖐 张开手掌 → 粒子绽放成花
- ✊ 握拳 → 聚成核心并旋转
- 🤏 拇指食指捏合 → 缩放
- ↔ 移动手 → 花朵跟随指尖

## 说明
- 纯本地运行，摄像头画面不上传。
- 手势识别 / Three.js 从 jsdelivr CDN 加载，需联网（国内被拦时开代理）。
